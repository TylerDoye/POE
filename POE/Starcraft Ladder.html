This application is focused on database managaement. the point was to make a program that utalised one database with a minimum of 3 tables.

I created the Battle.Net user data application, i has one form with three tabs user data,starcraft 2 and diablo 3. users can search for other players in the database. users can add or remove a user, add a serial key for a game activating the tab they may also remove a key deactivating the tab and removing information. reports can be generated depending on the game selected and the parameters for the report that will print out all users with the required stats
