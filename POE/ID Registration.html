This application will enables users to apply for certain documents at home affairs to fill out. The main focus of the assignment was event handling and debugging, which are found where reports and generating id. This application has 5 screens first being the login for users, then basic information screen. this screen takes all basic data that the users will have for all the forms, an id is generated and stores it in the base class. after filling out the form and clicking done they will be shown a report of what they filled in from all the data in the program
