Specs

This application enables users to interact with a file of information that can be randomely accessed. the user must be able to add a user. Once added the users are perminent and cannot be modified. the user can select to show a certain report depending on the type, the application must retrieve player details if they meet the criteria of the search, and all the players that fit the criteria will be shown on a report screen.
